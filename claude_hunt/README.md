# Shubham's AI/ML Job Hunt System

A comprehensive, systematic approach to job hunting that combines:
- Direct recruiting contact discovery
- LinkedIn authority building
- Personalized outreach
- Metrics-driven iteration

**Status**: Ready to execute immediately

---

## The Problem You're Solving

❌ 183 applications sent → 0% response rate

You're playing the wrong game. This system plays a different game.

---

## The Solution

A 4-pillar system that works in parallel:

1. **Contact Discovery** — Find hiring manager emails (not ATS queue)
2. **LinkedIn Authority** — Build visibility in AI/ML space
3. **Personalized Outreach** — Send targeted emails to real people
4. **Metrics Tracking** — Know what works, iterate, improve

**Expected outcome (8 weeks)**:
- 100-150 personalized emails sent
- 5-10% response rate (vs <1% from spray applying)
- 1-3 interview opportunities
- 500-1000 LinkedIn followers (high-signal audience)

---

## Quick Start (90 minutes)

1. **Read the strategy** (30 min)
   ```
   Start with: STRATEGY.md
   ```

2. **Find recruiting emails** (15 min)
   ```
   python3 contact_discovery.py
   → Creates: recruiting_contacts.csv
   ```

3. **Post on LinkedIn** (5 min)
   ```
   Copy first post from: linkedin_posts_ready.txt
   ```

4. **Send first email** (15 min)
   ```
   Use template from: outreach_templates.json
   Target: Modal or Together AI (good response rates)
   ```

5. **Log it** (5 min)
   ```
   python3 campaign_tracker.py
   ```

**You're done.** Repeat daily.
